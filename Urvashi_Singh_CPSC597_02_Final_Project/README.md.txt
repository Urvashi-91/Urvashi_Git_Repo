To run this Project, either run run.bat file or run 'module' on 'Covid19.py' file.
Wait for the program to load modules, in case of deprecation notice ignore. 

In case of error, please install below packages or as instructed in error message for missing modules.
pip install gensim
pip install pyLDAvis

Click on tabs to proceed as specified in Project report.

Facebook Posts should be in CSV file format.

PS: pyLDAvis graph takes time to process data, hence wait.
